BoDesk

BoDesk is an AI-powered platform designed to help homeowners plan and execute renovation and construction projects with confidence.

The goal of the project was to explore how AI can simplify the entire renovation process by combining project planning, cost estimation, material selection, visualization, and step-by-step guidance in a single platform.

Features

* AI-assisted project planning
* Cost estimation for renovation and construction projects
* Material and shopping list generation
* Step-by-step construction guidance
* AI-powered project visualization
* Project management dashboard
* Maintenance request management workflows

Example Use Case

A homeowner wants to build a deck.

Using BoDesk, the user can:

1. Describe the project and property conditions
2. Receive an estimated project cost
3. Generate a materials list
4. Get construction instructions tailored to the project
5. Visualize the finished result using AI image editing
6. Track project progress through the platform

Motivation

Many homeowners struggle to estimate costs, choose materials, understand construction requirements, and plan renovation projects effectively.

BoDesk was created to investigate how AI can reduce this complexity and make construction planning more accessible by providing practical guidance throughout the entire process.
